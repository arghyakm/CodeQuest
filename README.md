# Code Quest 🧠🚀

A standalone, interactive desktop application featuring dynamic state handling, randomized data structures, and custom UI event listeners. 

This application simulates a high-stakes, timed quiz environment, demonstrating complete control over application lifecycle management and object-oriented design.

## ⚙️ Core Architecture & Features
* **Algorithmic Randomization:** Utilizes advanced collection shuffling to dynamically pull 10 unique questions from a master bank every time a new session is initialized.
* **Lifecycle & State Transitions:** Seamlessly manages user progression through multiple application states (Login -> Rules -> Active Quiz -> Final Score).
* **Real-Time Processing:** Features an active background thread managing a 15-second countdown timer per question, triggering auto-submit protocols upon expiration.
* **Interactive UI/UX:** Built with a modern, flat-design philosophy incorporating custom interactive elements, hover animations, and coordinate-based layout positioning.

## 🚀 How to Run
1. Navigate to the **[Releases](../../releases)** section on the right side of this repository.
2. Download the latest `QuizApplication.jar` executable file.
3. Double-click the file to launch the application.
4. Enter your credentials, read the rules, and attempt to beat the clock!
